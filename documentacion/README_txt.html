    CLASE MATRIXDSL

    Es una clase diseñada para un lenguaje de dominio específico. Sus métodos sirven como puente hacia los

    métodos reales que se encuentran en la clase matriz, tanto matriz densa como matriz dispersa.

    Éstos actúan de forma que se pueda acceder a los métodos originales de la clase matriz mediante un lenguaje sencillo.

    Métodos:

    • operación

    • operando

    • ejecutar (permite completar la línea de código que se evaluará para llamar a los métodos originales)

    Las consultas a matrices que se quieran realizar se escribiran en el archivo matrices.rb, y para la ejecución de dichas consultas se compilará el archivo m.rb.
